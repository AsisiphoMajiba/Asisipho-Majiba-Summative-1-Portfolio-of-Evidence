# Bola's Beauty Salon Website

## Project Overview

Bola's Beauty Salon is a modern and responsive website designed to promote the services offered by Bola's Beauty Salon. The website provides customers with information about the salon, available beauty services, pricing, contact details, and an easy way to book appointments.

The website was developed using HTML, CSS, and JavaScript and follows modern web design principles to provide a professional and user-friendly experience.

---

## Features

### Homepage
- Welcome section
- Featured services

### About Us
- Salon history
- Mission and vision

### Services
- Hair Styling
- Brading/Conrows
- Nail Treatments
- Eyelash Extensions
- Facial Treatment

### Enquiry Page
- Appointment booking form
- Customer contact details collection

### Contact Page
- Contact form
- Business location
- Social media links

### Additional Features
- Responsive design for mobile and desktop devices
- Navigation menu with service dropdown
- Black, Gold, and White branding theme

## Installation

1. Clone the repository:

```bash
git clone https://github.com/AsisiphoMajiba/bolas-beauty-salon.git
```

2. Open the project folder.

3. Open `index.html` in your browser.
